# Security-Log-Analyzer
A Python-based cybersecurity tool that analyzes log files and detects suspicious events such as failed logins,invalid credentials, and system errors.

##Features

- Log file parsing
- Suspicious activity detection
- Failed login identification
- Error detection
- Security event counting

## Technologies Used

- Python
- Regular Expressions (Regex)
- File handling

## Example 

Input:
- sample_logs.txt

Output:
- Number of suspicious events
- Alert message for failed logins and errors
