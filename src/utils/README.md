Utilities

Purpose

This module contains helper functions that are used across the Security Log Analyzer project.

Instead of repeating the same code in multiple files, common functionality will be placed here.

Planned Utilities

- Logging
- Error Handling
- Configuration Loader
- File Validation
- Date and Time Utilities
- Common Helper Functions

Design Goal

Every utility should have a single responsibility and should be reusable by different modules.

Keeping common functions in one place makes the project easier to maintain and extend.
